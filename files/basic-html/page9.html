                                                布農族丹社群射耳祭暨
                                                傳統技藝競賽活動


                                                  103年4月12日
                                                  萬榮鄉馬遠國小
                                                  03-875-1321
                                                  萬榮鄉公所
                                                  每年4月中旬由花蓮萬榮鄉公所舉辦的「射耳祭」，為布農
                                                族人的傳統盛大祭典之一，其舉辦目的是要宣揚布農族的傳統
                                                與文化。整天的活動十分精彩，除了傳統的射箭比賽外，還可
                                                親眼目睹布農八部合音、剝箭筍比賽以及布農族的精彩舞蹈的
                                                活動，絕不可錯過。



                                                桐花風情 樂活鳳林

                                                  102年4月14日 14:00
                                                  鳳凰觀音寺
                                                  03-876-2771
                                                  鳳林鎮公所
                                                 2014年由鳳林鎮公所舉辦的桐花祭活動，將於溫暖的4月中在
                                                鳳林觀音寺的水源地廣場展開。今年的活動主旨，是帶領遊客登
                                                山健行，走步道、觀賞桐花，將運動及賞桐融為一體，身心兼
                                                顧，並有傳統客家歌舞表演與客家米食、傳統玩具的展示販售，
                                                爸媽可帶著孩子前來，以充滿客味的早晨拉開一天的序幕。



                                               2014 太魯閣春之頌


                                                  103年4月19日～103年5月31日
                                                  （每個週六，共計8天）
                                                  秀林鄉富世村等7部落
                                                  03-861-2116
                                                  http://www.shlin.gov.tw/
                                                  秀林鄉公所
                                                 春之頌為期8天的活動，主要是要帶領遊客於太魯閣境內的部
                                               落遊玩之餘，可以更深入的了解在地原民風情；走訪山林認識
                                               綠色生態，品嚐鮮美的箭筍，挖地瓜與傳統抓雞的體驗，每週
                                               都有不同的主題行程，十分多元豐富，詳細的活動內容，請洽
                                               秀林鄉公所。


                                               2014 花蓮翱翔季

                                                 103年7月5日起
                                                 鳳林環保科技園區
                                                 03-823-3506
                                                 花蓮縣政府觀光暨公共事務處
                                                 別再以為搭乘熱氣球是外國人才有的休閒活動！去（102）年
                                               花蓮首度舉辦熱氣球嘉年華，吸引大大小小的朋友來體驗、遊
                                               樂，色彩繽紛可愛的熱氣球，造型十分吸睛、討喜。目前花蓮翱
                                               翔季，已成為花蓮年度中的一大活動，看著它緩緩的升向天空，
                                               似乎可以預期，這個夏天會過得非常多彩多姿！
